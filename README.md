### Development

For development, it's convenient to use the Taskfile provided in the project. This file contains various commands that
can be executed using the `task` command.

See [taskfile.dev](https://taskfile.dev/) for more information on how to install and use it.

To see a list of available tasks, run:

```bash
task --list
```
